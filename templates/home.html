Welcome to QIM

Your centralized hub for automation and enhanced data tracking.

QIM (Quality Integrated Manager) is a Python Flask web portal that centralizes automation and enhanced data tracking for manufacturing and quality operations.
It integrates external services—such as Monday.com, Power BI, SQL, and Excel—into a unified dashboard, offering digital checklists, driver sign-in forms, inspection workflows, automated reporting, and digital-twin features.
The application is run by initializing a Flask app that sets up SQLite databases, registers routes for quality and inspection tools, and uses a thread pool to handle background tasks and Monday.com API interactions.

Developed by Tim Kim
